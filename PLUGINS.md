## 🛠️ **安装商业插件**

如果你购买了上述商业插件，按以下步骤安装：

### 方法1: Unity Asset Store
1. 打开Unity
2. Window > Package Manager > Asset Store
3. 搜索并购买插件
4. 点击导入到项目

### 方法2: 手动导入
1. 从官网下载 `.unitypackage` 文件
2. Unity > Assets > Import Package > Custom Package
3. 选择下载的文件并导入

### 方法3: Package Manager (Git URL)
某些插件支持通过Git URL安装：
1. Window > Package Manager
2. 点击左上角 `+` 号
3. 选择 "Add package from git URL"
4. 输入插件的Git仓库URL
